마우스 컨트롤러 클래스
-마우스 클릭 시 기준점 저장
-마우스 드래그 상태인지 체크
-드래그 상태일 경우 기준점과 좌표를 이용하여 플레이어 이동


플레이어 컨트롤러 클래스
-(무브 매개)좌표를 집어넣으면->트랜스폼의 좌표를 해당 좌표로 변경
-(라이프 매개)값을 집어넣으면->라이프 증감
-(스코어 매개)값을 집어넣으면->스코어 증감
-충돌 시(아이템 획득 매개)충돌 객체를 집어넣으면->충돌한 객체의 효과 온

무브 모델 클래스
-트랜스폼 세팅
-애리어 세팅
-좌표를 집어넣으면->트랜스폼의 좌표를 해당 좌표로 변경
1. (애리어 매개)애리어 내부에서만 이동
2. 애리어 무시하고 이동
-(애리어 매개)좌표를 집어넣으면->애리어 내부인지 리턴

애리어 모델 클래스
-스퀘어 세팅
-좌표를 집어넣으면->애리어 내부로 좌표 수정
-좌표를 집어넣으면->애리어 내부인지 리턴

-좌하, 우상 포인트 두 개 저장

포인트 구조체
-x, y 좌표값 저장

라이프 모델 클래스

-라이프에 변동이 있으면->(UI 매개)하트 갯수 증감
-라이프가 0 이하이면->(이벤트 매개)게임 오버 이벤트 호출


아이템 획득 컨트롤러 클래스
-(아이템 컨트롤러 매개)충돌 객체를 집어넣으면->충돌한 객체에 최상위 컨트롤러를 넘겨주며 효과 온
-(콤보 매개)콤보 갱신
1. 콤보 객체일 때 콤보 연장
2. 콤보 객체가 아닐 때 콤보 중단

콤보 모델 클래스
-콤보 연장
-콤보 중단
-콤보 시간 다 되면 중단
-현재 콤보 리턴

-(무브 매개)아이템 매니저에서 속도 값 가져와 무브 이용하여 아래로 이동
-(애리어 매개)애리어 밖으로 나갈 시(오브젝트 풀 매개)해당 아이템 릴리즈
